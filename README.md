# Airline-Ticket-Booking-UI
Airline Ticket Booking System
In this project, we have created a simple UI using visual force pages on Salesforce. The file in this repo is by extensions of .page 
Its only one file. It is to be copied and then executed on the Salesforce developer console after creating a visual force page there. Paste the code and click on preview. 
The Airline Ticket Booking UI System will be displayed on your side.
